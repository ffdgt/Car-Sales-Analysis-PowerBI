

### Power BI Car Sales Analysis Dashboard
This project is an interactive sales dashboard built with Power BI to analyze and track automotive sales performance for a dealership.

The main objective is to visualize key performance indicators (KPIs) and trends to enable data-driven decision-making.

##  Key Features & KPIs
Executive Overview: Tracks YTD/MTD Total Sales, YTD/MTD Average Price, and YTD/MTD Cars Sold.

Growth Tracking:Monitors Year-over-Year (YoY) growth and the absolute difference from the Previous Year-to-Date (PTYD).

Sales Trends: A weekly line chart shows sales velocity and identifies seasonal patterns.

Product Analysis: Donut charts break down YTD sales by vehicle Body Style and Color.

Geospatial Insights:An interactive map displays YTD cars sold by Dealer Region.

Brand Performance:A detailed grid ranks performance by Company (Brand), showing sales, volume, and average price.
